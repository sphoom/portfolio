A/B Testing for Website Search Interface Optimization

This project evaluates an A/B/n test conducted on a university library’s website to assess how different search interface designs impact user engagement. Using real-world data collected via Google Analytics, we analyze key metrics such as click-through rates (CTR), time spent on search results, and search success rates to determine the most effective design.

Statistical analysis, including ANOVA and post-hoc tests (Tukey’s HSD, t-tests), is applied to identify significant performance differences among variations. The insights from this study provide data-driven UX recommendations, guiding whether a new interface should replace the existing one. This project replicates real-world A/B testing methodologies used in UX and web optimization.
